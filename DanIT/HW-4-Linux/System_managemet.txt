1. Етапи завантаження системи Linux
   - BIOS/UEFI
   - Boot Loader
   - Kernel Initialization
   - Init Process

2. Перегляд системних журналів
   - Використовуючи команду `journalctl` для перегляду журналів systemd.
   - Перегляд журналів у папках `/var/log`, таких як `syslog`, `messages`, `auth.log`.

3. -rw-------
   Цей дозвіл означає, що файл доступний тільки власнику. Користувач може читати та писати у файл, але інші користувачі
   не мають жодного доступу до нього. chmod +x filename.

4. Різниця між apt і dpkg
   - apt - програма для встановлення пакетів в Ubuntu Linux.
   - dpkg - програма для встановлення, видалення, управління пакетами. Не вирішує автоматичні залежності.
